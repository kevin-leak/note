## js步骤

1. 创建一个核心对象 XMLHttpRequest

2. 编写回调函数

   ```js
   xmlhttp.onreadystatechange=function(){
   	if(xmlhttp.readyState==4 && xmlhttp.status==200){
   		alert(xmlhttp.responseText);
   	}
   }
   ```

3. open操作

   ```js
   xmlhttp.open("请求方式","请求路径");
   ```

4. send操作

   ```js
   xmlhttp.send([参数]);
   // post请求的参数放在send里面
   // 注意:若请求方式为post且有参数 必须设置一个头
   ```
